# TerraForm Tutorials

## Requirments
- terraform 
- aws-cli

## Environment Set

### Windows

```dos
    set AWS_ACCESS_KEY_ID=(AWS_ACCESS_KEY)
    set AWS_SECRET_ACCESS_KEY=(SECRET_ACCESS_KEY)
```

### macOS/Unix
```
    export AWS_ACCESS_KEY_ID=(AWS_ACCESS_KEY)
    export AWS_SECRET_ACCESS_KEY=(SECRET_ACCESS_KEY)
```
